# IMDB-Movie-Sales-Analysis

📊 Project Overview
This project analyzes movie sales data to uncover insights related to box office revenue, ratings, genres, and trends. The dashboard helps in understanding what drives movie success.

## 🎯 Objective
- Analyze box office performance of movies
- Identify top-performing genres and movies
- Evaluate relationship between ratings and revenue
- Track trends over time

## 🛠 Tools Used
- Power BI
- DAX
- Excel / CSV

## 📈 Key KPIs
- Total Revenue
- Average Revenue per Movie
- IMDb Ratings
- Total Votes

## 🔍 Key Insights
- High-budget and popular genre movies generate maximum revenue
- Certain genres consistently outperform others
- Movies with higher ratings tend to perform better at the box office
- Revenue trends vary significantly across years

## 📊 Features
- Interactive dashboard with filters (Year, Genre, Director)
- Top N movies analysis
- Revenue contribution by genre
- Trend analysis over time

## 💡 Business Impact
- Helps production houses identify profitable genres
- Supports decision-making for future investments
- Enables better understanding of audience preferences
